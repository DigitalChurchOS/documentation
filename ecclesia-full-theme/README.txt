# Ecclesia Full Church Theme Demo

This is a static multi-page demo of the Ecclesia default tenant church theme.

Pages included:
- Home
- About
- Sermons
- Events
- Ministries
- Prayer & Testimonies
- Giving
- Contact / Visit

The objective is to model a WordPress-style church starter theme where a tenant church activates the theme and mainly replaces copy, images, links, service times, pastor details, and ministry information.
