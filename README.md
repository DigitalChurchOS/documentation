# Digital Church Documentation

This branch contains the planning and architecture documentation for Digital Church OS before main project development begins.

## Contents

- `docs/platform-kernel-specification.md` - core platform kernel specification for shared entities, schemas, access controls, automation, and privacy foundations.
- `docs/churchos-prototype.html` - standalone HTML prototype for the project concept.
- `docs/assets/dark.png` - dark theme visual asset.
- `docs/assets/light.png` - light theme visual asset.

Development implementation should happen on the `main` branch. This branch is intended to stay focused on documentation and product planning.
